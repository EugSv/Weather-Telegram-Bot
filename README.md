# Weather Telegram Bot

Этот телеграм-бот позволяет получать текущую погоду в любом городе мира.

## Функционал
- Получение текущей погоды по названию города.
- Отображение температуры, минимальной и максимальной температуры, а также описания погоды.
- Возможность запросить погоду для другого города.

## Технологии
- **Python** — основной язык программирования.
- **pyTelegramBotAPI** — библиотека для создания Telegram-ботов.
- **OpenWeatherMap API** — API для получения данных о погоде.
